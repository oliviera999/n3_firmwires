# Rapports de Monitoring Système

Ce dossier contient les rapports de monitoring et de diagnostic du système en fonctionnement.

## Structure

- **reports/** - Rapports de monitoring et diagnostics

## Rapports Disponibles

- **ANALYSE_ORIGINE_PROBLEMES_CRITIQUES.md** - Analyse détaillée de l'origine des problèmes critiques (redémarrages, panics, mémoire, queue)

## Notes

- Les logs bruts (`.log.errors`) et analyses automatiques (`*_analysis.txt`) ne sont pas conservés car ils peuvent être régénérés
- Seuls les rapports d'analyse finaux sont conservés pour l'historique
